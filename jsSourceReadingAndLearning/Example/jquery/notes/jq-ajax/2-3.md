# 图片灯箱插件——lightBox 


该插件可以用圆角的方式展示选择中的图片，使用按钮查看上下张图片，在加载图片时自带进度条，还能以自动播放的方式浏览图片，调用格式如下：

```js
$(linkimage).lightBox({options})
```

其中 `linkimage` 参数为包含图片的 `<a>` 元素名称， `options` 为插件方法的配置对象。
